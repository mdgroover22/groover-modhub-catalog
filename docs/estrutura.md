# Estrutura do catálogo

## mods.json

Arquivo principal lido pelo Hub. Ele deve conter a lista de mods disponíveis.

## images/

Pasta para imagens dos mods.

Sugestão:

```txt
images/
└── man_tgx_e6_br/
    ├── cover.png
    ├── interior.png
    └── traseira.png
```

## changelogs/

Pasta para changelogs em Markdown.

Sugestão:

```txt
changelogs/
└── man_tgx_e6_br.md
```

## schemas/

Pasta com schemas JSON para validação futura do Publisher.

## templates/

Pasta com modelos de cadastro manual para testes iniciais.
